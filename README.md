# Sistema de Gestión de Inventario

Este proyecto es un Sistema de Gestión de Inventario desarrollado como parte de una actividad escolar.  
El sistema permite registrar, actualizar y consultar productos dentro de una base de datos.

El objetivo principal del proyecto es aplicar conceptos básicos de desarrollo web, bases de datos y programación integrando diferentes tecnologías.

---

## Tecnologías utilizadas

Este proyecto fue desarrollado utilizando las siguientes tecnologías:

- Python – Lógica del sistema
- MySQL – Base de datos para almacenar la información de los productos
- HTML – Estructura de las páginas web
- CSS – Estilos de la interfaz
- PHP – Conexión y manejo de datos entre la web y la base de datos

---

## Funcionalidades

El sistema incluye las siguientes funciones principales:

### Registro de productos
Permite agregar nuevos productos al inventario.

### Actualización de productos
Permite modificar la información de productos existentes.

### Consulta de productos
Permite visualizar todos los productos registrados en la base de datos.

---


## Objetivo del proyecto

El propósito de este proyecto es aprender a:

- Integrar frontend y backend
- Conectar aplicaciones con bases de datos MySQL
- Implementar operaciones CRUD (Create, Read, Update, Delete)
- Utilizar múltiples tecnologías en un mismo sistema
